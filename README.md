# PetStore API Automation Tests

Учебный проект по автоматизации тестирования REST API сервиса Swagger Petstore. Проект демонстрирует навыки написания автотестов на Java с использованием современных подходов к организации кода и валидации данных.

## Технологический стек
- **Язык:** Java 17
- **Сборщик:** Maven
- **Тестовый фреймворк:** JUnit 5
- **REST-клиент:** RestAssured
- **Библиотека проверок:** AssertJ (для читаемых ассертов)
- **Дополнительно:** Apache HttpClient (для именованных констант статусов), JSON

## Архитектура и ключевые практики
1. **Независимость тестов:** Использование `@BeforeEach` для генерации уникальных тестовых данных перед каждым запуском. Это гарантирует изоляцию тестов друг от друга.
2. **Валидация сложных структур:** Применение **Stream API** и работы с коллекциями (`List<Map>`) для поиска и проверки конкретных объектов внутри массивов ответов API.
3. **Чистый код (Clean Code):** Использование именованных констант (`HttpStatus.SC_OK`) вместо магических чисел, понятное именование методов.
4. **Покрытие сценариев:** Реализованы позитивные (CRUD операции) и негативные (обработка битого JSON, запрос к несуществующему ресурсу) сценарии.
